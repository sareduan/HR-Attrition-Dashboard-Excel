# HR Attrition Dashboard (Excel)

# Project Overview
This project analyzes employee attrition using 1,471 HR records and presents insights through an interactive Excel dashboard.

# Key Insights
- Total Employees vs Active Employees
- Attrition Rate (%)
- Attrition by Department
- Attrition by Age Group
- Attrition by Education Level
- Attrition by Job Role

# Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- KPI Cards
- Data Visualization
- Dashboard Design
- HR Analytics
